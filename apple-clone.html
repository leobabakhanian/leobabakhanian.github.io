Apple Clone

A clone of Apple.com built with React.

Live Code GitHub Link

Note: This was designed based on the 10/15/2021 version of Apple.com.

I challenged myself to recreate the main page of Apple.com in this project. I was able to recreate many of the elements precisely as they are on the real Apple website. There were only very minor details I was unable to recreate perfectly.

Technology used: HTML, CSS, JavaScript, NPM, React, MUI
